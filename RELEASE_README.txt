================================
  LOLHelper v2.1.1
  英雄联盟辅助工具
================================

📦 安装说明
-----------
1. 解压所有文件到任意文件夹
2. 双击运行 LOLHelper-WebUI.exe
3. 浏览器自动打开 http://localhost:5000
4. 开始使用！

✨ 主要功能
-----------
✅ 战绩查询 - 输入召唤师名称查看战绩
✅ 自动接受 - 匹配成功自动接受对局
✅ 敌我分析 - 选人阶段分析队友和敌人
✅ OP.GG集成 - 显示英雄Tier评级和胜率
✅ 实时游戏 - 查看当前游戏详细信息
✅ CV检测 - 游戏画面视觉识别（可选）

🎮 使用方法
-----------
1. 启动程序
   - 双击 LOLHelper-WebUI.exe
   - 等待浏览器自动打开

2. 连接游戏客户端
   - 确保英雄联盟客户端正在运行
   - 点击主页的"检测LCU"按钮
   - 看到"连接成功"即可使用

3. 查询战绩
   - 在"战绩查询"区域输入召唤师名称
   - 格式: 名称#Tag（例如: Hide on bush#KR1）
   - 点击"查看战绩详情"或按Enter键

4. 自动化功能
   - 点击"启用自动接受"- 匹配成功自动接受
   - 点击"启用敌我分析"- 选人时自动分析

⚙️ 系统要求
-----------
- 操作系统: Windows 10/11 (64位)
- 内存: 建议 8GB 以上
- 网络: 需要互联网连接
- 游戏: 英雄联盟客户端

⚠️ 注意事项
-----------
1. 杀毒软件提示
   - 首次运行可能被Windows Defender拦截
   - 这是正常现象（未签名程序）
   - 请点击"仍要运行"或添加到白名单

2. 端口占用
   - 程序使用 5000 端口
   - 如果端口被占用，请关闭其他程序

3. LCU连接
   - 必须先启动游戏客户端
   - 如果连接失败，请重启客户端后重试

4. 性能影响
   - 基础功能几乎无性能影响
   - CV检测功能会占用部分CPU资源

🔧 常见问题
-----------
Q: 程序无法启动？
A: 1) 检查是否有杀毒软件拦截
   2) 右键"以管理员身份运行"
   3) 确保所有文件完整解压

Q: 浏览器未自动打开？
A: 手动访问 http://localhost:5000

Q: LCU连接失败？
A: 1) 确保游戏客户端正在运行
   2) 重启游戏客户端
   3) 重启本程序

Q: 战绩查询失败？
A: 1) 检查召唤师名称格式（名称#Tag）
   2) 确保网络连接正常
   3) 该召唤师最近有游戏记录

Q: 占用内存过高？
A: 关闭CV检测功能可减少内存占用

📚 功能说明
-----------
1. 战绩查询系统
   - 支持所有服务器召唤师
   - 显示最近20场战绩
   - 包含KDA、胜率、英雄统计
   - 集成OP.GG Tier评级

2. 自动接受对局
   - 匹配成功0.5秒内自动接受
   - 后台持续监控
   - 可随时启用/停止

3. 敌我数据分析
   - 选人阶段自动识别队友
   - 游戏加载时识别敌人
   - 显示战绩、胜率、常用英雄
   - OP.GG数据集成

4. 实时游戏监控
   - 显示双方阵容
   - 实时KDA和装备
   - 经济对比
   - 战况分析

5. CV视觉检测（高级）
   - 英雄抬手识别
   - 小地图监控
   - 危险信号检测
   - 需要额外配置
